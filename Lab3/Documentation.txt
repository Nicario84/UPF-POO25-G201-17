En esta práctica se mantienen las clases básicas para manejar datos numéricos (Vector, Record, Dataset, StandardizedDataset) y se han introducido componentes para aprendizaje supervisado sencillo: Algorithm, Model, 
SupervisedLearner y TestLearner. El objetivo es poder entrenar un modelo lineal por descenso de gradiente sobre datasets de Records (entrada: Vector, salida: double) y validar el flujo desde los datos hasta las predicciones.

Para ello hemos creado la siguientes clases:
Vector, que representa un array de números y tiene métodos para hacer operaciones como sumar, restar, multiplicar, dividir, sacar la raíz cuadrada, calcular la norma, etc.
Record, que guarda un vector de entrada y una salida numérica.
Dataset, qe guarda una lista de Record y tiene métodos para calcular la media y desviación estándar de los datos. También tiene un método standardize que crea un nuevo conjunto de datos normalizado.
StandardizedDataset, que hereda de Dataset y guarda las medias y desviaciones estándar para poder transformar los datos.
TestDataset, que es la clase con el main, donde probamos que todo funciona bien.

Mantenemos las clases para datos: Vector, Record, Dataset y StandardizedDataset (normaliza con media y desviación).
Las nuevas clases para aprendizaje supervisado que hemos tenido que crear gan sido:
Algorithm, usa descenso de gradiente batch (learningRate, stoppingCriterion). Esto calcula gradiente y actualiza el modelo hasta converger.
Model, es un modelo lineal (params como Vector), predict(Vector) devuelve double (producto punto con input.augment()), update aplica el gradiente.
SupervisedLearner, que usa Algorithm y Dataset para entrenar y predecir.
TestLearner, es nustro main, tiene casos de prueba, imprime dataset, parámetros y predicciones.

Después del entrenamiento, params contiene los coeficientes del modelo lineal y las predicciones sobre nuevos inputs (producto punto con input.augment()) deben acercarse a las salidas observadas del dataset 
según la capacidad del modelo y la convergencia del descenso de gradiente.
